## Usar como extensão

Este repositório pode ser adicionado como **extensão** no MakeCode.

* abrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* clique em **Novo Projeto**
* clique em **Extensões** em baixo do menu com ícone de engrenagem
* procure por **https://github.com/marceloroberto/academia-mindmakers** e importe

## Editar este projeto ![Ícone de estado da compilação](https://github.com/marceloroberto/academia-mindmakers/workflows/MakeCode/badge.svg)

Para editar este repositório no MakeCode.

* abrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* clique em **Importar** e depois clique em **Importar URL**
* cole **https://github.com/marceloroberto/academia-mindmakers** e clique em importar
